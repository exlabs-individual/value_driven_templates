Sharing is caring.

Want to share your knowledge? You can learn here how.

Create the content

Inside src/content/articles directory create a new markdown file with the selected filename. Inside markdown file, You can write the content of Your article, remember article title, description, and image, will be added in the next step.

1

Add article metadata

Go to src/content/articles.json and add a new object with proper metadata for Your article, where fileName is just a plain name without extension of Your article content markdown file, imageUrl is a static file taken from the public directory or dynamic URL. Lastly, add Your nickname in the author field.

2

Describe yourself

If it is Your first time adding content, please go to src/content/authors.json and add a new entry without Your username used in article metadata, Your full name, and URL to Your avatar image.

3

Pull Request

That's it! Well done, now You can create a new Pull Request and wait for approval!
